I am Learning HTML

Who is this course for: Complete Beginners

Course for: Beginner
Course level: Introductory
Prerequisites: No prior knowledge of HTML required. We will take each concepts in detail and will understand the basic of building blocks of any web-page

What will I learn:

  • Modern, semantic and accessible HTML5
  • How to plan and design any website/ app
  • Basics tags/elements of HTML
  • Create complete HTML pages

Course content

  1. HTML syntax
  2. Tag, element and attribute
  3. HTML table tags
  4. Basic Structure of web-page
  5. HTML 5 and Semantics (div tags and named tag)
  6. HTML DOM
  7. Block and non-block tag
  8. HTML Body tags (header, nav aside, article, footer)
  9. Text Format and Decoration (H1-H6), p, em, b, a, strong
  10. OL, Unordered list

By the end of the course:

You will be able to create a simple web-page and understand the basic structure of any web-page.
